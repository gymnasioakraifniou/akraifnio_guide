# Ακραίφνιο Βοιωτίας – Τουριστικός Οδηγός

Τουριστικός οδηγός του χωριού Ακραίφνιο, φτιαγμένος από μαθητές.

## Δομή αρχείων

```
akraifnio-site/
├── index.html           ← Αρχική σελίδα (λίστα αξιοθέατων)
├── istoria.html         ← Ιστορία & Αρχαιολογία
├── thriskeftika.html    ← Θρησκευτικά Μνημεία
├── fisi.html            ← Φύση & Περιβάλλον
├── fagito.html          ← Φαγητό & Διασκέδαση
├── ekpaideysi.html      ← Εκπαίδευση & Υπηρεσίες
├── politismos.html      ← Πολιτισμός & Σύλλογοι
├── athlitismos.html     ← Αθλητισμός
├── akraifnio_index.docx ← Word έγγραφο για εκτύπωση με QR Codes
└── README.md
```

## Τοπική εκτέλεση

Απλά ανοίξτε το `index.html` σε οποιοδήποτε browser.  
Δεν χρειάζεται server ή εγκατάσταση.

## GitHub Pages

1. Ανεβάστε όλα τα αρχεία σε ένα GitHub repository
2. Πηγαίνετε: Settings → Pages → Source: Deploy from branch → main / root
3. Το site θα είναι διαθέσιμο στο: `https://[username].github.io/[repo-name]/`

## QR Codes (για το Word έγγραφο)

Για κάθε υποσελίδα δημιουργήστε QR Code στο:
- https://qr-code-generator.com
- https://goqr.me

URL μορφή: `https://[username].github.io/[repo-name]/istoria.html#akropoli`
